file in `content/blog/`. Drop a new `.md` file in that folder and it shows up
here automatically.

## How posts work

Each post is a Markdown file with frontmatter at the top:

```md
---
title: "My Post"
description: "A short summary."
date: "2025-06-15"
tags: ["tag-one", "tag-two"]
---
```

The frontmatter drives the listing, the page title, and the SEO metadata.

## Markdown features

You get everything from [GitHub Flavored Markdown](https://github.github.com/gfm/):

- Bullet lists
- **Bold** and _italic_ text
- `inline code`
- [Links](https://nextjs.org)
