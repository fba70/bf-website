If you manage a lot of real estate objects you normally have a problem how to create the content which attracts attention of potential clients. And the complexity and costs grow rapidly going from text descriptions to graphics and finally to videos. Still we know that video content “sells” the best.

What if there is a way to use your real estate property descriptions and existing pictures (which you normally already have on your web site) and automatically create good quality short video for each of your objects? ***At any scale you need and up to 100-times cheaper than doing it manually*****.**

All you need to do is get data from your web site / CMS with real estate objects information and configure storytelling templates which will define the logic and design of the video content. You can use your property-related texts, pictures, existing video fragments, audio files as well as your brand identification elements (logos, audio tracks, background pictures, intro/outro videos).

To make more attractive standard animation (motion, rotation, scaling, etc.) can be used as well as specific AI-augmented effects such as automated translation, text-to-speech conversion, graphical elements transformation or generation and so on.

You have a lot of flexibility to define the rules how all these video elements and effects will play depending on you data, for example:

Property typeProperty parametersGeo locationLocal agent contactsLanguage of the countryAnd any other dependencies which you can define.

Geo location can be used to extract and show automatically in the videos nearby attractions, map of the neighbourhood etc. Charts and graphs can be added as well. Want to show several properties in one video (for example same type or similar geo location) - it’s also possible to do defining corresponding storytelling templates.

Finally, video can be formatted to fit requirements of the target platform (web site, social media, digital signage displayed, etc.).

IN4COM GmbH offers data-driven automated video content production platform **AVICA** ([https://avica.cloud](https://avica.cloud)) which allows you to create videos at any scale and with costs up to 100-times cheaper compared to manual work. AVICA supports scheduled automatic updates of the content based on defined changes in source data (for example product price change).
